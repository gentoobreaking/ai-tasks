## 描述

現狀：儀表板只顯示「當前」狀態。
優化：建立一個 `stats.py`，分析 `updated` 日期的變動。

## 已實作功能

### stats.py
- 計算過去 7 天 / 30 天完成的任務數量
- 計算週期時間（平均、中位數）
- 生成 Mermaid xychart-beta 燃盡圖（過去 14 天趨勢）
- 顯示本週 vs 上週完成趨勢

### 整合
- 已整合到 `update_projects.py`
- 每次執行會自動在 PROJECTS.md 頂部顯示效能分析

## 使用方式

```bash
# 单独运行 stats.py
python3 scripts/stats.py

# 更新 PROJECTS.md（包含效能分析）
python3 scripts/update_projects.py
```

## 產出

- 新增 `scripts/stats.py`
- 修改後的 `scripts/update_projects.py`
- PROJECTS.md 現在包含效能分析區塊
