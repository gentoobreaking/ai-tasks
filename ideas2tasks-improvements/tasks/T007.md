# T007 - Board Status 自動同步

## 目標
加入 Board 的 Issue 沒有設定 Status 欄位（Pending / In Progress / Done / Skipped），導致 Board 上狀態與本地任務狀態不一致。修改 executor.py 在加入 Board 時自動設定 Status，並在 cleanup/sync-board 時校正現有項目。

## 驗收標準
- [x] 新建立的 Issue 在 Board 上顯示正確的 Status
- [x] 執行 `--cleanup-github` 後，done/skip 的 Board Status 同步更新
- [x] 執行 `--sync-board` 時校正現有項目的 Status

## 修改內容
- 新增 `_update_board_item_status()` 與 Board 常數
- `sync_single_task_to_github()`: 加入 Board 後設定 Status
- `_create_github_issue()`: 同上
- `cleanup_github_issues()`: 關閉 Issue 時同步 Board Status
- `sync_board()`: 新增項目時根據 issue state 設定 Status
