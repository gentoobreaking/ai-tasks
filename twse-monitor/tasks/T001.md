## 目標
建立 `twse_monitor.py` 核心框架，含以下監控模組：

### 模組一：個股每日重大訊息（M1）
- **Endpoint**: `GET /opendata/t187ap04_L`
- **頻率**: 每日 1-2 次（盤後 17:00 後）
- **邏輯**: 抓取全市場重大訊息，篩選豪關注的股票代碼，變動時通知

### 模組二：除權除息預告（M2）
- **Endpoint**: `GET /exchangeReport/TWT48U_ALL`
- **頻率**: 每日
- **邏輯**: 抓取全市場預告，篩選關注股票，通知除權除息日期

### 模組三：殖利率 / 本益比 / 股價淨值比（M3）
- **Endpoint**: `GET /exchangeReport/BWIBBU_ALL`
- **頻率**: 每日
- **邏輯**: 抓取全市場本益比/殖利率/淨值比，篩選並記錄；僅變動時通知

## v1.1 新增（2026-05-06 上午）

### 1. 每股股價監控閾值
```json
{
  "watchlist": ["0050", "2330", "00981A"],
  "thresholds": {
    "2330": {
      "max_price":  null,      "min_price":  null,
      "max_pct_up": null,      "max_pct_down": null,
      "circuit_up": true,      "circuit_down": true,
      "circuit_pct": 10
    }
  }
}
```
- **通知分級**：`circuit` = 🔴緊急（iOS/watchOS 響鈴），`threshold` = 🟡警告，`percent` = 🔵參考

### 2. Apple Watch / iPhone 推送
- Telegram Bot API 原生支援 iOS/watchOS，`disable_notification: False` 即響鈴
- 緊急訊息：`send_telegram(urgent=True)` 強制響鈴
