---

## 功能一：設定持有成本 `--cost`

**用法：**
```bash
python3 twse_monitor.py --cost 2330 2150
python3 twse_monitor.py --cost 0050 88.5
```

**行為：**
- 更新 `stocks` 表的 `cost` 欄位
- 若股票不在 watchlist，先 INSERT 基本資料
- 成功後印出格式化結果

**輸出範例：**
```
✅ 持有成本已更新
   股票：2330 台積電
   成本：2150.00
   現價：2250.00（close_today）
   未實現損益：+100.00（+4.65%）
```

---

## 功能二：查詢 DB `--show-db`

**用法：**
```bash
python3 twse_monitor.py --show-db            # 全部表
python3 twse_monitor.py --show-db --table stocks   # 只看 stocks
python3 twse_monitor.py --show-db --table seen_items # 只看 seen_items
```

**輸出：**
- ASCII 表格格式化輸出（等寬對齊）
- 各表筆數統計
- `stocks` 表：代碼 / 名稱 / 持有成本 / 今日收盤 / 昨日收盤 / 未實現損益
- `seen_items` 表：類別 / 鍵 / 內容摘要 / 時間

---

## 功能三：查詢設定檔 `--show-config`

**用法：**
```bash
python3 twse_monitor.py --show-config
```

**輸出：**
- 分段格式化輸出
- watchlist 列表
- 各股票閾值設定（表格）
- Telegram 設定狀態（Token 隱碼顯示）
