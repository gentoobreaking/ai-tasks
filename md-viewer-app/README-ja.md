# md-viewer-app

**md-viewer** は、macOS 用に特別に構築された高性能 Markdown リーダーです。

## プロジェクトの構成

|プロジェクト |テクノロジースタック |ステータス |
|------|------|------|
| `md-viewer-webview/` | Go + webview_go + Swift-markdown (CGO) | ✅ **メインライン** |
| `md-viewer-fyne/` |ゴー+ファイン |コントラストの維持 |

**webview ソリューション**が最終的に選択されました。[T008 比較評価](tasks/T008-專案分裂與對比評估.md) を参照してください。
