# приложение md-viewer

**md-viewer** — это высокопроизводительная программа чтения Markdown, специально созданная для macOS.

## Состав проекта

| Проекты | Стек технологий | Статус |
|------|---------|------|
| `md-viewer-webview/` | Go + webview_go + Swift-markdown (CGO) | ✅ **Основная** |
| `md-viewer-fyne/` | Го + Файн | Сохранение контраста |

Наконец было выбрано **решение веб-просмотра**, см. [T008 Сравнительная оценка](tasks/T008-專案分裂與對比評估.md).

## Функция реализована

| Функция | Задача |
|------|-------|
| Механизм анализа Swift-markdown | Т003 |
| 6 тем мгновенного переключения | Т005, Т014 |
| подсветка синтаксиса Highlight.js + кнопка копирования | Т017 |
| Панель настройки (масштаб/шрифт/язык/номер строки) | T011-C~G, T011-FIX |
| Родное меню NSMenu | Т011-А, Т011-Б |
| i18n (традиционный китайский/упрощенный китайский/английский/японский/корейский) | Т011-ФИКС-02, Т011-ФИКС-03 |
| Перетащите, чтобы открыть файл .md | Т010 |
| Автоматическая перезагрузка мониторинга файлов (включая перезагрузку флэш-памяти) | Т013 |
| ассоциация файлов macOS (дважды щелкните .md, чтобы открыть) | Т011-Л |
| Функция перевода (⌘⇧T, 3 бэкэнда) | Т028 |
| Экспорт HTML/PDF | Т019 |
| Полноэкранный режим (⌘F) | Т011-ФИКС-04 |
| Сохранение масштаба (сохраняется при перезагрузке/перезапуске) | Т011-ФИКС-01, Т011-ФИКС-05 |
