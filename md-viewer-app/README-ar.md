# تطبيق MD-Viewer

**md-viewer** هو قارئ Markdown عالي الأداء مصمم خصيصًا لنظام التشغيل macOS.

## تكوين المشروع

| المشاريع | مكدس التكنولوجيا | الحالة |
|------|---------|-----|
| `md-viewer-webview/` | اذهب + webview_go + Swift-markdown (CGO) | ✅ **الخط الرئيسي** |
| `md-viewer-fyne/` | اذهب + فاين | الحفاظ على التباين |

تم أخيرًا تحديد **حل عرض الويب**، راجع [T008 التقييم المقارن](tasks/T008-專案分裂與對比評估.md).
