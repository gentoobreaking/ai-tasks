## 需求描述
將 Markdown 文件內容翻譯成不同語言，並保留 Markdown 格式（標題、程式碼區塊、內聯程式碼）。

---

## 技術方案

### 支援 3 種翻譯後端

| 後端 | 環境變數 | 每日額度 | 備註 |
|------|----------|----------|------|
| **MyMemory（預設）** | `TRANSLATE_BACKEND=mymemory` | 1000 字/天 | 免費，無需 key |
| **DeepL** | `TRANSLATE_BACKEND=deepl` + `DEEPL_API_KEY` | 50 萬字/月（免費 tier） | 需 API Key |
| **LibreTranslate** | `TRANSLATE_BACKEND=libre` + `LIBRETRANSLATE_API_KEY` | 視自架實例 | 可搭配自架實例，URL 可透過 `LIBRETRANSLATE_URL` 自訂 |

### API 端點
- **MyMemory**：`https://api.mymemory.translated.net/get?q=<text>&langpair=<from>|<to>`
- **DeepL**：`https://api-free.deepl.com/v2/translate`（或 `api.deepl.com`）
- **LibreTranslate**：`https://libretranslate.com/translate`（或自架 URL）

### Markdown 格式保留邏輯
- 標題前綴（`#`, `##`, …）不翻譯，譯文前綴原標題
- 程式碼區塊（` ``` `）整段跳過不翻譯
- 內聯程式碼（`` ` ``）跳過不翻譯
- 原文格式（如 `**粗體**`、`[連結](url)`）在譯文後重建

## 主要程式碼片段

### translator.go — 後端分派
```go
func Translate(text, sourceLang, targetLang string) (string, error) {
    backend := os.Getenv("TRANSLATE_BACKEND")
    switch backend {
    case "deepl":
        return DeepLTranslate(text, sourceLang, targetLang)
    case "libre":
        return LibreTranslate(text, sourceLang, targetLang)
    default: // mymemory
        return MyMemoryTranslate(text, sourceLang, targetLang)
    }
}
```

### main.go — mdTranslate binding
```go
wv.Bind("mdTranslate", func(html, sourceLang, targetLang, callback string) {
    go func() {
        defer func() { recover() }()
        result, err := TranslateMarkdown(html, sourceLang, targetLang)
        safeResult := strings.ReplaceAll(result, "`", "\`")
        wv.Dispatch(func() {
            wv.Eval(fmt.Sprintf("window.%s && window.%s(`%s`)", callback, callback, safeResult))
        })
    }()
})
```

## 使用方式

```bash
# 預設 MyMemory（無需設定）
./md-viewer

# 指定 DeepL
TRANSLATE_BACKEND=deepl DEEPL_API_KEY=xxx ./md-viewer

# 指定 LibreTranslate（可自架）
TRANSLATE_BACKEND=libre LIBRETRANSLATE_API_KEY=xxx ./md-viewer

# 自訂 LibreTranslate URL
TRANSLATE_BACKEND=libre LIBRETRANSLATE_API_KEY=xxx LIBRETRANSLATE_URL=https://my-instance.com/translate ./md-viewer
```

## 已知限制

- MyMemory 每日 1000 字額度，額度用盡後會返回錯誤（尚無 UI 提示）
- MyMemory 不支援 `source=auto`，自動映射為 `en`
- 翻譯結果以 goroutine 非同步執行，UI 不會卡住
