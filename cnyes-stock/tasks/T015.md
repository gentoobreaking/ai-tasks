# T015 - Shebang 與可移植性改善

## 目標
讓腳本不依賴特定虛擬環境路徑即可執行，提高可移植性。

## 驗收標準
- [x] Shebang 改為 `#!/usr/bin/env python3`
- [x] 文件或 README 說明使用者需自行安裝 `playwright` 依賴
- [x] 保留 venv 作為建議執行方式，但不強制綁定
- [x] `--system-chrome` 依賴 `CHROME_SYSTEM` 常數路徑，與 venv 無關，不受影響

## 備註
- 絕對路徑 shebang 只在該特定 macOS 使用者環境有效
- 改為 `env python3` 後，腳本會使用 PATH 中的 Python 3
