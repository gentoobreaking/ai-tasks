# T009 - 瀏覽器資源管理改善

## 目標
修正 Playwright browser 未明確關閉的問題，確保例外發生時瀏覽器行程不會外洩。

## 驗收標準
- [x] `fetch_news` 與 `fetch_trending_tw_stock` 中的 browser 改用 context manager（`with p.chromium.launch() as browser:`）
- [x] page 物件也在 context manager 中管理（`with browser.new_page() as page:`）
- [ ] 模擬網路中斷等異常情境，確認 browser 能被正確關閉
- [x] 正常爬取流程不受影響

## 備註
- 目前寫法：`browser = p.chromium.launch(...)`，若中間拋出例外則 browser 不會被 cleanup
- Python context manager 是最乾淨的做法：`with p.chromium.launch() as browser:`
