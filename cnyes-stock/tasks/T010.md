# T010 - 參數解析重構 — 改用 argparse

## 目標
將手動 `sys.argv` 解析改為 `argparse.ArgumentParser`，消除全域 `CHROME` 變數，並加入環境變數支援。

## 驗收標準
- [x] 使用 `argparse` 取代 `sys.argv` 手動解析
- [x] 同時支援 `--date=YYYY-MM-DD` 與 `--date YYYY-MM-DD` 兩種格式
- [x] 提供 `--help` 說明所有參數
- [x] 移除 `global CHROME`，瀏覽器模式改為函式參數 `use_system_chrome`
- [x] 確認 `--telegram`、`--trending`、`--system-chrome` 行為不變
- [x] `load_telegram_config()` 改為只讀環境變數 `TELEGRAM_BOT_TOKEN` / `TELEGRAM_CHAT_ID`
- [x] 移除 `gold_monitor_config.json` 所有參照（含 `TELEGRAM_CONFIG` 常數與 `import json`）
- [x] `IDEAS2TASKS_TASKS_DIR` 環境變數已列於 docstring
- [x] 文件字串（docstring）更新為環境變數說明

## 備註
- 目前 `--date YYYY-MM-DD`（無等號）會無聲地忽略，因為只檢查了 `startswith("--date=")`
- `global CHROME` 讓測試與重構變得困難
- 環境變數優先順序：ENV > JSON > 預設值
